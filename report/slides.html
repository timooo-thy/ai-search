04 - IMPLEMENTATION

Precision Retrieval

Semantic Intent (Dense)

Captures conceptual intent (e.g., "handling user logout") using bge-large-en-v1.5. Bridges terminology gaps between user and code.

Keyword Precision (Sparse)

BM25 indexing ensures exact identifier matching (e.g., "AuthMiddleware") which semantic vectors can often dilute.

Distributed-Based Score Fusion (DBSF) normalises scores across disparate retrieval methods.

05 - VISUALISATION

Structural Context

CodeOrient transforms retrieved files into an interactive Dependency Graph:

  • Nodes: Language-aware code entities (Functions/Classes).
  • Edges: Real import/call relationships extracted via AST.
